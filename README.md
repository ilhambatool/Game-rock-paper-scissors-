# Rock Paper Scissors

An interactive Rock Paper Scissors game built with **HTML**, **CSS**, and **JavaScript**. Play against the computer, with the first player to reach **5 points** winning the game.

## Features

- 🎮 Interactive browser-based gameplay
- 🤖 Random computer opponent
- 📊 Live score tracking
- 📝 Displays both players' choices each round
- 🏆 Announces the winner when 5 points are reached
- 🔄 Play Again functionality

## Live Demo

🔗 https://ilhambatool.github.io/Game-rock-paper-scissors-/

## Project Evolution

This project was originally implemented as a **console-based** game and was later upgraded to a fully interactive **DOM-based** browser application.
